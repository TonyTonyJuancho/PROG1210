Jane Doe Customer Help:

1- In the Details Page you can see all the data from the customer you select in the index.


2- The first table represent all the repair request the customer has and the actual status for each request.


3- The second table represent all the equipment the customer has brought to the store, you can see the equiment name and model.


4- You can start the repair process for the qeuipment just by clicking on respective "Select" for the equipment.

5- In the bottom you have 3 buttons: "back" throws you to the customer index page, "edit customer" redirects you to the page where you can edit the customer and "delete customer" gives you the option to delete a customer in case that it was created by mistake.
