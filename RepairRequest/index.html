Repair Requests Note: Repair requests are required for repairs to be added to the system Reapir Request Help:

1- In the Repair Request Page you can see the ID,Date, Reason, and status from the repairs requests


2- You can click on any repair request box and you will see all of their details and a link under the details for the technicians to create a repair if it's not staryted yet.


3- The first table is the repair table where you will be able to go to the details of the repair if it has been created already.


4- The second table is the customer table, you can see some of the information about the customer and if it's needed you can go to the customer page using the "Go to Customer" Link.
